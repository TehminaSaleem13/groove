Instructions to setup Shipworks to import orders in Groovepacker

  1. 1. Create an action in Shipworks

    Open ShipWorks & click on Manage > Actions > New Action (name the new action).

  2. 2. Setup the action

    Under Actions, Select "An order is downloaded"

    From the Add Task dropdown Select External > Send web request

    For template: Select "XML Source"

  3. 3. Integrate with Groovepacker

    Copy (ctrl-C) and Paste (ctrl-V) your Request URL to the ShipWorks Request URL blank. Here is your Request URL:

    {{stores.single.request_url}}{{stores.single.auth_token}}

  4. Setup is complete!

    Now, when orders are downloaded into ShipWorks they will be automatically imported into GroovePacker. Setting the ShipWorks store to Inactive in GroovePacker will disable the import.
